🚀 TalentBridge - Job Seeker & Employer 
PortalTalentBridge is a modern, full-featured job portal designed to connect talented professionals with top employers. The platform provides a seamless experience for browsing opportunities, tracking applications, and managing recruitment pipelines.
✨ Key Features
Dual-Portal System: Dedicated, customized interfaces for both Job Seekers and Employers.
Job Discovery: Advanced search functionality to filter jobs by title, location, and type (Remote/Full-time).
Application Tracking: Real-time dashboard for candidates to monitor "Applied," "Accepted," and "Pending" statuses.
Recruitment Tools: A robust Employer Dashboard to post new jobs, manage listings, and review incoming applications.
Profile Management: Comprehensive profile editing for users and detailed branding options for company profiles.
Modern UI: A clean, professional, and responsive design optimized for a high-quality user experience.

🛠️ Tech Stack
Frontend:HTML5, CSS3, JavaScript (React.js)
Styling: Tailwind CSS / Custom CSS
Icons: Lucide React
